## How to Preprocess?

- 지문 세그멘테이션 (Fingerprint Segmentation): 지문 이미지에서 지문 영역을 분리하는 과정으로, 이 과정을 통해 지문 영역 외의 노이즈나 배경 영역을 제거하고, 정확한 지문 영역을 추출할 수 있습니다.

- 지문 이진화 (Fingerprint Binarization): 지문 이미지를 이진화하여, 픽셀 값을 0 또는 1로 만드는 과정으로, 이 과정을 통해 지문 영역에서 특징점이나 라인 등을 쉽게 추출할 수 있습니다.

- 지문 보정 (Fingerprint Enhancement): 지문 영상의 품질을 향상시키기 위한 기술로, 주로 지문 상의 라인 정보를 강화하여 라인과 같은 지문 특징점의 검출을 용이하게 합니다.

- 지문 정규화 (Fingerprint Normalization): 다양한 지문 이미지의 크기, 방향, 회전 등을 표준화하여 비교 분석을 용이하게 하는 과정입니다.

- 지문 필터링 (Fingerprint Filtering): 지문 이미지의 노이즈나 구석 등의 불필요한 영역을 제거하여 지문 영역을 더욱 명확하게 추출하는 기술입니다.


- 필터링(Filtering): 이미지에서 노이즈를 제거하거나, 선명도를 향상시키는 등의 목적으로 사용됩니다. 대표적인 필터로는 가우시안 필터, 미디언 필터 등이 있습니다.

- 엣지 검출(Edge Detection): 이미지에서 경계 부분을 찾아내는 것으로, Canny, Sobel, Prewitt 등의 알고리즘이 사용됩니다.

- 전처리 기반 특징 추출(Preprocessing-based Feature Extraction): 지문 이미지에서 지문의 형태를 나타내는 경계선이나 고유한 질감 패턴을 추출하기 위해 필요한 전처리 과정입니다. 예를 들어, 지문 이미지를 정규화하거나, 지문의 곡률을 보정하는 과정 등이 이에 해당합니다.

- 딥러닝 기반 전처리(DL-based Preprocessing): 딥러닝에서는 자동으로 feature를 추출하도록 하는 경우가 많기 때문에, 전처리 과정도 자동으로 처리될 수 있습니다. 예를 들어, CNN에서는 이미지를 입력으로 넣으면 자동으로 feature map이 생성되며, 이를 통해 전처리 과정이 수행될 수 있습니다.

- 데이터 증강(Data Augmentation): 지문 이미지를 다양한 각도나 크기로 변환하여 데이터의 양을 늘리는 기술입니다. 데이터 증강을 통해 모델이 더욱 일반화된 특징을 추출할 수 있습니다. 대표적인 데이터 증강 방법으로는 회전, 크기 조정, 좌우 반전, 가우시안 노이즈 추가 등이 있습니다.

각 지문 특징 추출의 종류에 따라 필요한 전처리 기술들이 조금씩 다를 수 있지만, 위에서 언급한 전처리 기술들은 대부분의 경우 유용하게 사용될 수 있습니다.

Bimodal image는 이미지에서 뚜렷한 두 개의 픽셀값 집합이 존재하는 경우를 의미합니다. 즉, 이미지에서 두 가지 다른 패턴 또는 객체가 있는 경우를 말합니다.

예를 들어, 흑백 이미지에서 어두운 부분과 밝은 부분이 뚜렷하게 구분되어 있고, 이 두 부분이 서로 다른 특징을 가지고 있는 경우 bimodal image라고 할 수 있습니다. 이러한 경우, 이미지 처리나 컴퓨터 비전 분야에서는 이러한 두 개의 집합을 분리하고 분석하는 기술이 많이 활용됩니다. 이를 통해 이미지에서 두 개의 서로 다른 객체나 패턴을 자동으로 인식하고 추출할 수 있습니다.
------------- 지문의 특징점
지문은 사람마다 다르게 구성되어 있기 때문에, 각각의 지문은 고유한 특징을 가지고 있습니다. 이러한 특징을 추출하여 지문 인식 시스템에서 사용할 수 있습니다. 주요 지문 특징은 다음과 같습니다.

미세한 선명도 변화: 지문은 미세한 선명도 변화로 이루어져 있습니다. 지문에서 미세한 변화를 추출하여 지문 인식 시스템에서 사용됩니다.

지문 끝점(Ending Point): 지문 끝점은 지문 끝에서 끝나는 선분을 말합니다. 지문 끝점은 지문을 추적하는 데 사용됩니다.

지문 분기점(Bifurcation Point): 지문 분기점은 지문에서 두 개 이상의 선분으로 나누어지는 지점을 말합니다. 지문 분기점은 지문의 분기점을 추적하는 데 사용됩니다.

삼각 지점(Delta Point): 삼각 지점은 지문의 삼각 형태를 이루는 지점을 말합니다. 지문의 삼각 지점은 지문 인식 시스템에서 사용됩니다.

지문 골절(Lineament): 지문 골절은 지문의 선상에서 분기점과 끝점을 연결하는 선분을 말합니다. 지문 골절은 지문 인식 시스템에서 지문의 일부로 사용됩니다.

이러한 지문 특징을 추출하고 저장하여 지문 인식 시스템에서 사용됩니다. 지문 인식 시스템에서는 사용자가 제공한 지문의 특징을 데이터베이스에 저장하고, 그 다음에는 인식할 지문의 특징을 추출하여 저장된 데이터베이스의 지문 특징과 일치하는지를 확인합니다.

--- sauvola 알고리즘 설명
Sauvola 이진화는 지역 이진화(Local Binarization)의 하나로, Sauvola와 Pietikäinen이 2000년에 발표한 논문에서 제안된 알고리즘입니다.

Sauvola 이진화는 이미지의 각 픽셀을 이진화할 때 해당 픽셀 주변의 픽셀들의 밝기 값을 고려합니다. 이 알고리즘은 지역적인 임계값(threshold)을 계산하여 이진화를 수행합니다.

알고리즘의 계산 과정은 다음과 같습니다.

이미지에서 지정한 크기의 윈도우(Window)를 이동하면서 각 픽셀을 중심으로 잡습니다.
윈도우 내의 모든 픽셀의 평균값과 표준편차를 계산합니다.
임계값 T(x,y)는 다음과 같은 수식을 이용하여 계산됩니다.
T(x,y) = μ(x,y) * (1 + k * ((σ(x,y) / R) - 1))
여기서, μ(x,y)는 윈도우 내의 모든 픽셀 값의 평균, σ(x,y)는 표준 편차, k는 사용자가 지정한 값이며, R은 이미지에서 사용되는 동일한 값입니다.
임계값 T(x,y)를 기준으로 해당 픽셀을 0 또는 1로 이진화합니다.
Sauvola 이진화는 밝기가 조금 어두운 이미지에서도 잘 작동하며, 이미지의 크기가 작거나, 노이즈가 많은 경우에도 유효합니다. 이 알고리즘은 특히 문서 스캔 등의 응용 분야에서 많이 사용됩니다.

----------손실된 세부정보를 얻는법
일반적으로, 이진화는 이미지 데이터의 노이즈를 제거하고 영역 분할 및 특징 추출과 같은 다양한 이미지 처리 알고리즘에 필요한 전처리 과정으로 많이 사용됩니다. 이진화를 통해 데이터 크기를 줄이고 빠른 처리 속도를 얻을 수 있기 때문입니다.

그러나 이진화를 수행하는 과정에서 세부 정보가 손실될 수 있습니다. 이는 일반적으로 희소한 노이즈나 작은 특징과 같은 이미지 데이터의 작은 변화를 놓치는 것과 관련이 있습니다. 이러한 손실된 세부 정보는 육안으로 확인하기 어렵거나 불가능할 수 있습니다.

따라서, 이진화의 효과를 평가하고 선택한 이진화 방법이 손실된 세부 정보를 최소화할 수 있는지 판단하기 위해서는 해당 이미지에 대한 전문적인 분석이 필요합니다. 예를 들어, 지문 데이터에 대한 이진화 방법을 평가할 때에는, 세부 정보가 얼마나 잘 보존되었는지에 대한 정량적인 분석을 수행할 수 있습니다. 이를 위해, 예를 들어, 윤곽선의 근사화, 끝점, 지점의 수, 지문 특징 포함의 개수 등과 같은 특징을 추출하고 비교하는 방법을 사용할 수 있습니다.

------ 히스토그램으로 비교하기
이 코드는 fingerprint.jpg라는 이미지를 이진화하는 여러 방법을 비교하고 시각적으로 확인할 수 있습니다. BINARY, BINARY_INV, TRUNC, TOZERO, TOZERO_INV 등의 다양한 이진화 방법을 사용하여 처리한 이미지와 각각의 히스토그램을 출력합니다.

비교한 결과를 통해, 각 이진화 방법이 지문 데이터에 어떤 영향을 미치는지 확인할 수 있습니다. 이를 통해 손실되는 세부 정보를 최소화할 수 있는 이진화 방법을 선택할 수 있습니다.
